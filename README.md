## Fake Letter

Generate fake letters (.pdf) from markdown (.md)

## Available variables

- User
- Email
- ZipCode
- City
- Phone
- Acount
- Date1
- Date2

## Installation

```
brew install pandoc
brew cask install mactex
dotnet install tool -g wk.FakeLetter
```

## Usage

```
wk-fake-letter <TEMPLATE.md> <COUNT>
```
